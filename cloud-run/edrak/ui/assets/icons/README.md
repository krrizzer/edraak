# App icons

All icons are optional — the app shows graceful placeholders until a file exists.
Use square PNGs (transparent background looks best).

| File | Used as |
|---|---|
| `Amad.png` | The Edraak app logo (login screen, home header, PWA icon) |
| `ALINMA.png` | مصرف الإنماء logo (host bank row, bank panels) |
| `ALRAJHI.png` | مصرف الراجحي logo |
| `SNB.png` | البنك الأهلي السعودي logo |
| `RIYAD.png` | بنك الرياض logo |
| `SAB.png` | البنك السعودي الأول logo |

File names are case-sensitive and must match exactly (bank codes in CAPS).
After adding files, restart `flutter run` (hot reload doesn't pick up new assets).
